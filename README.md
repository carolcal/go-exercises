# Go Exercises – Exercism Solutions

This repository contains my solutions to Go (Golang) exercises from [Exercism](https://exercism.org/profiles/carolcal).
I created it to practice Go, explore idiomatic patterns, and track my progress while learning the language.

---

## My Journey Through Go

Here’s the path I followed on Exercism to learn Go, covering topics from the very basics to more advanced concepts.

Each node represents an exercise I completed and a concept I practiced — from variables, loops, and slices to interfaces, methods, and error handling.
In this repository, each folder represents a single exercise.

---

## 📘 About Exercism

[Exercism](https://exercism.org/profiles/carolcal) is a platform that provides hands-on programming exercises in multiple languages.
Each exercise focuses on a specific concept — from basic syntax and control flow to advanced topics like concurrency and interfaces.
